# Hice-Magia.
 GameLab2023-HiceMagia

- Hice Magia – Juego 2D Point & Click (Unity)

Hice Magia es un juego 2D estilo point and click desarrollado en Unity con C#, donde el jugador debe combinar ingredientes en un caldero para obtener la mayor cantidad de puntos antes de que se acabe el tiempo.

Descripción del proyecto:
- El juego se basa en la experimentación y toma de decisiones rápidas.
- El jugador selecciona ingredientes e intenta descubrir combinaciones correctas que suman puntos, mientras evita aquellas que penalizan el puntaje. Todo ocurre bajo un tiempo límite, lo que agrega dificultad y ritmo al gameplay.

Mecánicas principales:
- Juego 2D point and click
- Sistema de combinaciones de ingredientes
- Caldero interactivo como núcleo del gameplay
- Sistema de puntuación
- Combinaciones incorrectas con penalización
- Tiempo límite para completar la partida
- Interacción simple e intuitiva

Cómo ejecutar el proyecto:
1) Clonar el repositorio: git clone https://github.com/NehuenPey/HiceMagia.git
2) Abrir Unity Hub.
3) Seleccionar Open Project y elegir la carpeta del proyecto.
4) Abrir la escena principal.
5) Ejecutar el juego desde el editor (Play).
